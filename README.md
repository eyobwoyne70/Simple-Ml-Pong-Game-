# ML Pong Game

A simple Pong game where a machine learning model plays against you! The AI learns to predict the paddle position based on the ball's X and Y coordinates.

## 📋 Project Overview

This project demonstrates how to use a basic machine learning model to predict game mechanics in real-time. The model takes the ball's position as input and outputs the predicted paddle position.

## 🎮 How It Works

1. **Game**: Classic Pong game with you vs AI
2. **ML Model**: Trained on ball position data (ball_x, ball_y)
3. **Prediction**: Model predicts where the paddle should be to hit the ball
4. **Result**: AI plays against you automatically!
